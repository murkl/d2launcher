Example Configuration

d2_dir="$HOME/.local/apps/diablo2"
d2_exec="Diablo II.exe"
d2_args="-3dfx"
wine5_enabled=true
gui_width=400
gui_height=250
desktop_icon=true
update_check=true
mxl_update_check=true
logging_enabled=true

Diablo II Arguments

These are available arguments you can use in d2launcher.conf.

Enable Glide wrapper -3dfx
Window mode -w
Skip to Median XL login -skiptobnet
No sound -ns

D2 Stats

Open D2Stats before start Diablo II because the script looking for a running D2Stats.exe and executes sigma-loader.exe instead of Diablo II.exe.

Issues

In case of your d2_binary_dir is inside the d2launcher dir (or sub dir), the sigma-loader.exe crashes on start. Workaround is to move the d2_binary_dir outside the d2launcher dir.
